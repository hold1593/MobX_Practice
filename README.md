# Mobx 라이브러리를 사용한 미니 TodoList

### package version
```json
"dependencies": {
  "mobx": "^6.3.9",
  "mobx-react": "^7.2.1",
  "react": "^17.0.2",
  "typescript": "^4.5.4"
}

"devDependencies": {
  "@babel/plugin-proposal-class-properties": "^7.16.5",
  "@babel/plugin-proposal-decorators": "^7.16.5"
}
```

### 구현된 기능
1. Login기능
2. 유저별 Todo list 보기
3. Todo list 추가 및 삭제
4. Rating 수정
5. mobx를 이용하여 상태를 전역으로 관리하며 새로운 기능을 추가할 수 있습니다 😋
